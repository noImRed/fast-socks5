# fast-socks5
a simple SOCKS5 server written in Luvit

[Luvit](https://luvit.io)

## what's currently working:
- commands: 
   1. [x] CONNECT
   2. [ ] BIND
   3. [ ] UDP ASSOCIATE
- auth methods:
   1. [x] NO AUTH
   2. [x] USERNAME/PASSWORD
   3. [ ] GSSAPI
- addresses:
   1. [x] IPv4
   2. [x] domain
   3. [ ] IPv6
